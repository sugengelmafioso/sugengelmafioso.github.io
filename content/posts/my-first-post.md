This is the first post on my new Hugo-powered blog!

## What's New

I've just migrated this site to use **Hugo** with the **hugo-coder** theme, hosted on **GitHub Pages** via GitHub Actions.

### Why Hugo?

- Fast builds (seconds, not minutes)
- Static HTML output (perfect for GitHub Pages)
- Minimal dependencies (single binary!)
- Great theming system
- Markdown-first workflow

- AI agents & LLM applications
- DevOps & CI/CD workflows
